# Google Sheet 컬럼 매핑 (HR AI Agent 설문)

설문 응답이 저장될 때 사용하는 **열 이름(헤더)** 목록입니다.  
긴 질문은 번호/약어로 매핑했습니다.

---

## 공통
| 열 이름 | 설명 |
|--------|------|
| `survey_id` | 설문 고유 ID (제출 시 자동 생성) |
| `timestamp` | 제출 일시 (ISO 형식) |

---

## Part 1 – 기본 정보
| 열 이름 | 설문 문항 |
|--------|-----------|
| `P1_title` | 제안 과제명 |
| `P1_dept` | 제안 부서 |
| `P1_contact` | 담당자 |
| `P1_user` | 최종 사용자 (User) |

## Part 2 – 종합 및 판정
| 열 이름 | 설문 문항 |
|--------|-----------|
| `score_value` | 비즈니스 임팩트 총점 (0~15) |
| `score_feasibility` | 실행 가능성 총점 (0~20) |

---

## 시트 설정 시 참고
- 첫 행에 위 열 이름을 **순서대로** 헤더로 넣으면 됩니다.
- 체크박스는 `1`(예) / `0`(아니오) 또는 `Y`/`N`으로 저장해도 됩니다.
- 여러 줄 텍스트는 시트에서 줄바꿈으로 저장하거나, JSON 배열로 저장할 수 있습니다.
